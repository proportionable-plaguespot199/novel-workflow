# AI辅助小说创作实战指南 (小说焚决)

**前言**

加入社区有一段时间了，在不断的交流互动下终于达到了3级。这里给各位佬友们分享一个兼职赚钱的方法——每天只需花费2-3小时甚至更短的时间来写小说。我本人主要利用谷歌网页版、AI Studio、Claude (CLI) 以及 Cursor (CX) 等AI工具来辅助创作。目前写了三个月，因为同时开了好几本，总收入已经有一万多。

如果觉得不急着用钱，且缺乏耐心去打磨长篇，我十分推荐**短篇小说**，以数量堆积取胜（详见第一章）。切记：不管AI再怎么强大，生成的内容有时仍会带点机械感，写出来后我们依然需要人工进行审阅和润色。

以下是我个人的经验浅谈，希望对各位佬友有用。

## 第一章：网文变现经验与平台选择

### 1. 市场调研与平台推荐

- **切忌盲目开书**：动笔前务必了解你准备发书的网站上，目前最火的是什么类型。不看市场盲目写，很容易扑街。
- **平台选择**：首推**番茄**和**飞卢**。起点对新手AI写手不太推荐。建议先用番茄与飞卢练手，等成绩起来了，你对“如何利用AI结合写小说”也熟练了，再考虑更高的平台。
    - *番茄稿费说明*：以同人文为例，稿费主要看每日阅读人数（千次阅读转化几十块钱），此外每个月达成字数还有600-800元的全勤奖。

### 2. 热门题材推荐

- **长篇同人/玄幻**：番茄上推荐写《海贼王》（比如最近很火的莫利亚与光月御田热度极高）、《人民的名义》、港综、火影、一人之下、西游等。不过同人单价较低。想提高收益可以尝试玄幻（如遮天风格，像《横推诸天从荒古第1帝开始》很吃香）、规则怪谈、诡异游戏（非常容易被改编）、以及最近很火的“749局”相关题材。
- **男频爆款人设**：杀伐果断、道德底线灵活、乐子人、不圣母、不会见了美女就走不动路。

### 3. 短篇小说打法（以量取胜）

如果嫌长篇太耗时，可以走短篇路线，在番茄上很多此类文都是AI写的。

- **女频反套路（暴打奇葩）**：现在很多女频文套路是“男主角被男小三陷害 -> 女主折磨男主（掏心掏肺/献祭亲人） -> 男主死心说不爱了 -> 女主幡然醒悟报复男小三并重新追求男主”。针对这种套路，我们可以写“不窝囊、直接虐女主和男小三”的反套路爽文。
- **其他短篇题材**：性转题材、纯爱剧情、女性独立搞事业、反抗原生家庭等。

## 第二章：AI辅助创作工作流

下面以我在番茄写**港综同人文**为例，拆解具体的AI创作步骤。

### 第1步：语料收集与风格提炼

不要为了让AI学习就自己去通读全文。去番茄和起点下载那些追读人数多、写得好的同类小说，用文件切割工具（如 Calibre Portable）切成几部分。

将文本丢进 Claude (CLI) 或 Cursor (CX)，让它**提炼总结风格与写法**，并找出这些小说的缺点以便我们避坑。将提炼出的风格文件保存好。

### 第2步：设定小说背景与金手指

在 CLI 与 CX 中多次对话修改，确定以下内容直到满意：

- 故事背景是什么时候？
- 主角身份是什么？
- 金手指（系统/外挂）是什么？

> **提示**：一定要在系统指令中强调“正文禁止用大量环境描写、华丽词藻、大量比喻修辞来充字数”。除非写的是特定玄幻文，否则AI极容易生成大量且油腻的比喻。
> 

### 第3步：资料搜集与融合

- **影视融合**：确定融入哪些电影和电视剧。比如写港综，可以融入《无间道》三部曲、《夺帅》、《杀破狼》、《追龙》、《五亿探长雷洛传》、《警察故事》系列、《古惑仔》系列等。推荐背景设在80年代或以后。可以利用谷歌、Grok等工具直接帮你列出清单。
- **深度研究**：用谷歌深度研究去生成现实资料，比如对应年代的经济、军火、黑道、产业等相关知识，作为现实血肉补充。

### 第4步：大纲打磨与问题排查

先写大纲，不满意就丢进 CX 里让它找出逻辑问题；把问题复制到 CLI 让它修改；修改完再丢回 CX 检查，直到两个AI都确认无逻辑漏洞。

## 第三章：核心创作原则与技巧

### 1. 核心创作法则

- **“Show, don’t tell” (展示，不要讲述)**：用细节堆砌真实，用行动证明强大。不要用旁白宣告情感。
- **“盐溶于汤”叙事法**：主角的野心和价值观不能光靠口号喊出来，必须内化于其每一个行为中。
- **“梗的艺术”**：将网络热梗提炼内核，用符合小说年代语境的方式说出来（如“用最正经的语气说最荒诞的话”）。
- **全员智商在线**：配角不是工具人，必须有自己的算盘和反击。主角的强大在于压服聪明人，而不是碾压傻子。
- **三七开的日常剧情**：万物皆为“饵”。日常的每一笔（占30%）都必须成为未来主线冲突（占70%）的伏笔或钩子，严禁写成流水账。

### 2. 塑造代入感的六大支柱

1. **基础信息标签化**：让读者一句话就能认识主角（谁、在哪、发生了什么、有什么特征）。
2. **具体化与熟悉感**：描写必须具象到读者日常能接触到的事物，用“接地气”的场景拉近距离。
3. **认知与情绪共鸣**：让主角做出读者也会做的选择。利用亲情纽带（如相依为命的奶奶被欺负）或不公待遇（如被恶意克扣工资）制造强情绪缺口。
4. **欲望与期待感**：网络小说的本质是满足欲望（权势、逆袭）。作者要主动制造“憋屈”的情绪缺口，引导出读者对“扬眉吐气”的强烈期待。
5. **五感代入**：加入视觉、听觉、嗅觉、触觉、味觉的细节（如潮湿黏背的短袖、医院消毒水味），增强画面感。
6. **人设与反差**：不要十全十美的人设。完美的角色需要接地气的小缺点（如霸道总裁下班爱吃淀粉肠），这样角色才像活人。

### 3. 增强代入感的实用技巧

- 利用熟悉的物品和口头禅。
- 给角色打上牢固且一致的标签（痣、习惯性动作、穿搭）。
- 制造“冲突的紧迫性”，用危机感推着读者走（参考灾难电影）。
- **减少“了”字及转折词**（如虽然、但是、却）。用多了会给读者一种“作者在强行说教”的排斥感。

## 第四章：网文创作常见“雷区”与排雷指南

在创作和审校AI稿件时，务必对照以下常见问题进行排查：

### 1. 开篇与包装问题

- **书名与简介赶客**：书名过于文艺无核心信息（如单字“慕”），或者简介与正文脱节。
    - *对策*：书名采用“题材+核心爽点+主角行为”格式；简介300字内做到“抛困境+亮外挂+留悬念”。
- **开篇拖沓/信息轰炸**：前三章疯狂堆砌世界观和背景，迟迟不进主线。
    - *对策*：遵循黄金三章法则（第一章抛冲突，第二章展金手指，第三章定小目标）。信息要在互动中分层植入。

### 2. 剧情与节奏问题

- **流水账叙事**：详写“吃饭、坐公交”等无意义日常。
    - *对策*：加入“前因后果”与“强情绪”。比如同样是赶车，加入“被催债、被老板骂、手机掉下水道”的倒霉前置，就能将流水账变成紧凑的情绪铺垫。
- **主线模糊/偏离**：支线太多掩盖了核心目标。
    - *对策*：每3-5章设置小目标，严格遵循“1条主线+1条支线”结构，删减所有不推进主线的情节。
- **过渡生硬/视角混乱**：场景频繁无逻辑跳转；第一人称和第三人称混用。
    - *对策*：确定一个主要视角贯穿场景，过渡时用人物动作或心理变化进行衔接。

### 3. 冲突与爽点问题

- **冲突乏力/金手指失衡**：主角遇到困难轻松秒杀（金手指过强），或金手指毫无卵用。
    - *对策*：给金手指设置“使用限制”或“代价”；让敌我双方有来有回，制造势均力敌的压迫感。
- **爽点延迟或突兀**：主角被嘲讽后3章才反击（延迟）；或者无铺垫莫名其妙的高潮（突兀）。
    - *对策*：“小冲突→快解决→强反馈”；高潮前必须埋下伏笔，高潮后必须展现带来的收益和改变。

### 4. 人设与情感问题

- **人设扁平与前后矛盾**：所有人说话一个腔调；前期大女主后期变圣母；反派降智送人头。
    - *对策*：每次行动前反问“这符合他的利益和过往性格吗？”；通过“核心标签+反差细节”塑造立体感。
- **情感生硬（AI味重）**：两人前一秒互砍，下一秒强行宣誓结盟；末世绝境中强行写校园风恋爱。
    - *对策*：一切关系转变必须有“事件驱动”（如共同御敌），用微动作代替直白的“他很感动”。

### 5. 文笔与排版问题

- **刻意秀文笔/辞藻堆砌**：用生僻词和华丽排比写网文。
    - *对策*：网文重在通俗易懂，一个精准的形容词胜过十个华丽的废话。
- **排版密密麻麻**：段落过长，手机端阅读吃力。
    - *对策*：手机阅读视角下，每段控制在3-5行以内，多换行，留出呼吸感。

## 第五章：小说核心设定框架（防偏离约束）

在将指令输入AI前，必须建立一套不可违背的核心设定准则（以港综黑道/玄幻为例）：

### 1. 主角与外挂设定

- **主角性格**：极致利己 + 有底线。容忍黑暗但绝不圣母，每一个举动背后必有利益算计。斩草除根，借力打力，阳谋无解。
- **系统功能**：需明确名称、作用、商城机制及空间功能，且必须伴随主角成长。

### 2. 创作底线约束（严禁事项）

- **严禁机械降神**：系统不能凭空变出原子弹，一切金手指产出必须符合小说世界逻辑。
- **严禁反派降智**：主角必须赢在信息差和手段更狠，绝不是因为对手蠢。
- **严禁时间线错乱**：严格遵守真实历史或既定世界线的人生死活。
- **严禁无脑后宫**：女性角色必须有血有肉有私心，决不能是只会喊“666”的花瓶。
- **严禁数据模糊**：金融做空赚了多少？走私利润几个点？数据必须精确。
- **严禁文青病**：大战前夕不要突然感慨人生，直接干！
- **严禁设定吃书**：空间设定多大就是多大，除非有升级过程。
- **严禁不懂装懂**：涉及枪械射速、卡壳率、金融杠杆做空，专业术语必须准确，通过联网查证。
- **严禁AI式说教**：绝不要在文末总结“这个故事告诉我们……”。

## 语言去油与口号约束
优先使用动作、器物反应、局部感官与具体判断制造压迫感，不用陈词滥调硬顶气氛。
对“冷笑”“蝼蚁”“轰然炸裂”“倒吸一口凉气”“瞳孔骤缩”“满场死寂”等高疲劳词保持克制。单章中同一高识别词默认只允许出现 1 次。
群像反应不要一律写成“全场震惊”，改写成 1-2 个具体角色的身体反应或判断偏差。反派的狠话必须贴合身份，不要复制通用口号。审稿时主动检查词汇疲劳。

## 混合任务编排与改写边界
- 审查 + 改写：先列问题，再给最小修法，再输出修改结果。
- 设定 + 试写：先定设定边界，再给试写片段。
- 卷纲 + 正文：先给卷纲或节拍表，再写正文，避免正文反吃大纲。
- 润色：只改表达、节奏，不改事实。
- 改写：可改叙述顺序、画面，保留核心事实。
- 重写：可重构场景推进，但不改主设定和大事件结果。
- 续写：只在现有文本之后向前推进，不反改前文。

## 正文创作流程
1. **建立实时上下文**：先读 `99_当前状态卡.md`，锁定当前时间点、资源和伏笔，找到最近相关章节建立最小连续上下文。
2. **校验动机、信息边界与收益**：动笔前先自问：此刻利益最大化的选择是什么？谁先动手？反派有何诉求和误判？本章收益能落到具体数值吗？如果答不上来，先补动机链。
3. **识别章节类型**：判断是战斗章（重画面与收益）、布局章（重试探与威慑）、过渡章（重状态变化与钩子）还是回收章（优先回应旧伏笔）。
4. **施加场景压力**：用动作、伤势、声音、血腥味落地“强”。每个场景至少推进一项：信息、地位、资源、伤亡、仇恨、境界。小冲突尽快兑现反馈。核心对手必须有脑子。
5. **执行外显一致性检查**：生成正文前，输出表格版【写作自检】；在正文后追加表格版【章节结算】并同步更新状态卡与账本。
