# 都市禁止模式

> 都市题材特有的负面清单。与 base-rules.md 中的全局禁止模式叠加使用。

---

## 叙事禁止

| 禁止模式 | 说明 | 替代方案 |
|----------|------|----------|
| 龙王赘婿 | 主角身份揭露后全员跪拜 | 身份是工具不是结局，揭露后仍有博弈 |
| 打脸流水线 | 看不起->打脸->再看不起->再打脸 | 冲突升级要有实质变化 |
| 无限提款机 | 金钱来源不明，随用随有 | 每笔资金有来源和成本 |
| 主角光环 | 所有偶然都对主角有利 | 成功靠信息和资源，不靠运气 |
| 法律儿戏 | 随意操纵法律或无视法律后果 | 法律是约束也是武器，有代价 |

## 角色禁止

| 禁止模式 | 说明 | 替代方案 |
|----------|------|----------|
| 脸谱化阶层 | 富人都坏、穷人都善 | 每个阶层都有好人坏人 |
| 女性花瓶 | 女性角色只负责崇拜主角 | 女性角色有独立立场和能力 |
| 反派纸糊 | 反派只会嚣张然后被打脸 | 反派有资源有手段有反扑 |
| 万能主角 | 主角什么都会 | 主角有短板，需要团队和资源 |

## 设定禁止

| 禁止模式 | 说明 | 替代方案 |
|----------|------|----------|
| 虚假法律 | 编造不存在的法律条文 | 查证后引用，或回避具体条文 |
| 商业胡扯 | 不合逻辑的商业操作 | 商业逻辑需经得起推敲 |
| 物价穿帮 | 物价水平不符合现实 | 重要金额查证 |
| 技术魔法 | 把技术当魔法用 | 技术能力有边界 |
