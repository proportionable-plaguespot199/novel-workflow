## 角色使用规则

### 调用方式

在写作或审查过程中，当遇到特定领域问题时，切换到对应专家视角：

```
[切换视角: 数值审计员]
本次吞噬天骄本源，增量是否合理？与上一个同档样本相比是否一致？
```

### 使用约束

- 每次最多同时激活 **2 个** 专家视角
- 专家视角只提供领域判断，不替代主写手的叙事决策
- 专家之间的意见冲突由 Orchestrator 仲裁
- 不要为了展示专业知识而在正文中堆砌设定

### 常见组合

| 场景 | 推荐组合 |
|------|----------|
| 战斗章 | 战斗编舞师 + 爽点工程师 |
| 突破章 | 境界设计师 + 数值审计员 |
| 吞噬结算 | 数值审计员 + 法宝锻造师 |
| 势力交锋 | 势力棋手 + 反派设计师 |
| 新地图 | 地图绘制员 + 氛围渲染师 |
| 种族冲突 | 种族学者 + 杀伐裁判 |
| 伏笔回收 | 伏笔编织者 + 爽点工程师 |
