# 玄幻风格圣经

> 修仙/玄幻叙事风格定义。暴力美学、黑暗宇宙、极端掠夺、强因果反馈。本文件是玄幻题材的写作基调总纲。

---

## 核心气质

- **高压迫感**：每个场景都有生存压力、资源争夺、势力碾压
- **高反馈**：三章内必须有可感知的收益兑现或代价支付
- **强因果**：所有能力提升和资源获取都要有可追溯的来源
- **暴力美学**：战斗不是数值对撞，是力量的物理表现和身体的具体反应
- **掠夺感**：吞噬、掠夺、压榨是核心玩法，每次战斗都有可量化的收益

---

## 数值系统

### 核心原则

- 所有资源变动必须具体到**数值**，不用"暴涨""海量""难以估量"
- 同档同类资源的增量不得无说明跨越一个数量级
- 超过既有样本 3 倍需解释原因，超过 10 倍视为异常
- 重复吞噬同质资源必须写明衰减，不默认全额

### 数值记录

每次涉及收益的章节，必须在【章节结算】中记录：

| 字段 | 说明 |
|------|------|
| 期初值 | 战前/吞噬前的当前数值 |
| 增量 | 本次获得的具体增量 |
| 增量来源 | 具体的吞噬对象/资源名称 |
| 增量依据 | 为什么是这个数值（境界、完整度、稀有性） |
| 期末值 | 战后/吞噬后的当前数值 |
| 折损说明 | 如有衰减，说明原因 |

## 境界与战力

### 表现差异化

不同境界的战斗描写必须有**质的区别**：

| 境界层次 | 战斗特征 | 描写重点 |
|----------|----------|----------|
| 低阶 | 肉搏、法术对轰 | 力量、速度、伤口、血腥 |
| 中阶 | 领域碰撞、法则初显 | 空间扭曲、自然异象、压制感 |
| 高阶 | 法则对抗、天地异变 | 规则碰撞、因果反噬、代价 |
| 顶级 | 超越常规 | 概念层面的对抗 |

### 战力锚点

- 每次突破必须有前文铺垫的因果链
- 不能临时发明设定解决战斗
- 败退时必须有合理的生存逻辑，不能靠运气

## 叙事原则

### 场景压力

- 用动作、伤势、声音、重量、冲击、温度、血腥味落地"强"
- 每个场景至少推进一项：信息、地位、资源、伤亡、仇恨、境界
- 小冲突尽快兑现反馈，不把爽点无限后置

### 战斗描写

- 动作要有**受力感**：拳砸在胸口肋骨断裂的声音，不是"一拳打飞"
- 伤势要**具体**：不写"身受重伤"，写"左臂骨折、三根肋骨断裂、法力枯竭"
- 收益要**量化**：不写"实力暴涨"，写"+3000 微粒，来源：对方天骄本源"

### 反派智商

- 核心对手必须有脑子：有试探、有误判、有反扑
- 反派基于自己的已知信息行动，不能全知
- 边缘敌人不必人人演满一套，但不能故意给主角让路

## 感官锚点

| 感官 | 玄幻示例 |
|------|----------|
| 视觉 | 法则纹路、血色雾气、空间裂缝、古阵光芒 |
| 听觉 | 骨骼碎裂声、法则共鸣、虚空震荡、心跳 |
| 触觉 | 法力冲击的实体感、温度骤变、肌肉撕裂 |
| 嗅觉 | 鲜血铁锈味、焦糊、药香、腐朽 |
| 味觉 | 喉间血腥、丹药回甘、苦涩 |

## 语言风格

### 基调

- 冷酷、凌厉、不废话
- 短句为主打节奏，长句铺压迫
- 动词密度高，形容词克制
- 不要文艺腔，不要鸡汤，不要感悟

### 禁止词汇模式

- 高疲劳词单章只允许 1 次：冷笑、蝼蚁、轰然炸裂、倒吸凉气、瞳孔骤缩、满场死寂
- 群像反应不写"全场震惊"，改写具体角色的具体反应
- 反派狠话不用通用口号，贴身份和处境

---

## 主角约束（如适用）

- 杀伐果断，斩草除根
- 可以示弱、钓鱼、借刀杀人，但前提是利益更大
- 绝不心软、绝不圣母、绝不留手
- 无女主路线；女性角色可以重要，但有立场有手段，不做情绪挂件
- 非功能性时刻保留：受伤后的喘息、独处时的思考、意外的犹豫

---

## 风格污染防御

正文、试写、改写必须锁定玄幻小说风格，禁止混入：
- 都市腔、科幻腔、游戏系统播报腔
- 轻小说吐槽腔、影视剧本腔
- 历史演义腔、散文抒情腔
- 任何与黑暗玄幻不符的风格
