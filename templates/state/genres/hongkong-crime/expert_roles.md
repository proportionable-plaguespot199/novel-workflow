## 角色使用规则

### 调用方式

在写作或审查过程中，当遇到特定领域问题时，切换到对应专家视角：

```
[切换视角: 老差骨]
这场抓捕行动的程序合不合规？便衣行动需要谁批准？
```

### 使用约束

- 每次最多同时激活 **2 个** 专家视角
- 专家视角只提供领域判断，不替代主写手的叙事决策
- 专家之间的意见冲突由 Orchestrator 仲裁
- 不要为了展示专业知识而在正文中堆砌术语

### 常见组合

| 场景 | 推荐组合 |
|------|----------|
| 警匪冲突 | 老差骨 + 黑社会顾问 |
| 商业布局 | 金融操盘手 + 地产经纪 |
| 法律攻防 | 大律师 + 廉政调查员 |
| 年代校准 | 香港史学者 + 粤语语言学家 |
| 底层场景 | 九龙城寨居民 + 茶餐厅老板 |
| 灰色交易 | 走私船长 + 夜总会妈咪 |
