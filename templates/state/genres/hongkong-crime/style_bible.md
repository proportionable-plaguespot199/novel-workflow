# 港综风格圣经

> 80-90 年代港式叙事风格定义。本文件是港综题材的写作基调总纲。

---

## 时代基调

### 核心气质

- **江湖气**：讲义气但更讲利益，人情债和商业逻辑并行
- **制度摩擦**：殖民末期的权力真空、黑白灰地带、多方势力博弈
- **时代厚重感**：物价、建筑、交通、通讯方式、社会风气都要落到 80-90 年代
- **人情世故**：面子、排场、规矩、辈分、地头意识

### 感官锚点

| 感官 | 80-90 年代港风示例 |
|------|-------------------|
| 视觉 | 霓虹灯、唐楼天台、茶餐厅玻璃柜、马场看台、维港渡轮 |
| 听觉 | 粤语电台、麻将声、茶楼叫卖、警笛、码头汽笛 |
| 嗅觉 | 凉茶铺药味、大排档油烟、海水咸腥、雪茄/香烟、发廊洗发水 |
| 触觉 | 湿热天气的黏腻、点钞手感、握手力度、西装质感 |
| 味觉 | 茶餐厅奶茶、叉烧饭、威士忌、街边鱼蛋 |

## 叙事原则

### 智斗 > 武斗

- 核心冲突靠利益博弈推进，不靠拳头
- 打架是手段不是目的，每次动手都要有明确的利益计算
- 威慑、借势、联盟、背叛、信息差才是主要武器

### 钱权落地

- 每笔交易要有具体金额、条件、风险
- 地位变化通过他人反应兑现：谁开始点头哈腰，谁不再敢直呼其名
- 资源变化通过物件体现：车、表、房子、铺面、码头份额

### 灰度空间

- 没有纯善纯恶，每个人都有自己的算盘
- 警察不全是好人，古惑仔不全是坏人
- 利益面前，昨天的兄弟今天可能反水

### 场景切入

- 不用背景介绍开场，用场景切入：
  - 一个交易正在进行
  - 一个电话打断了饭局
  - 一个人站在天台抽烟
  - 一辆车停在某个不该停的地方

## 语言风格

### 台词规则

| 角色类型 | 语言特征 |
|----------|----------|
| 大佬 | 话少、点到即止、用沉默施压、偶尔粗口但有分量 |
| 古惑仔 | 直接、粗糙、带地域俚语、但不是满嘴脏话 |
| 白道（警察/律师） | 表面文雅、潜台词多、威胁藏在规则里 |
| 商人 | 精明、数字精确、永远在算账、人情也是生意 |
| 女性角色 | 有立场、有手段、不是花瓶、可以比男人更狠更聪明 |

### 叙述语言

- 简洁硬朗，不要文艺腔
- 短句为主，长句用于铺垫压迫感
- 动词驱动，少用形容词
- 环境描写服务于气氛，不是风景画

### 港味分寸

- 适度使用粤语词汇增加质感，但不要每句都塞
- 可用的：收皮、扑街、靓仔、饮茶、揸fit人、开片、劈友
- 慎用的：过于地道的俚语需要上下文让读者理解
- 禁止的：为了"港味"硬塞不合语境的粤语

## 冲突模型

### 标准博弈结构

```
利益冲突产生
    |
    v
信息收集（谁知道什么，谁不知道什么）
    |
    v
试探/威慑（摸底对方筹码和底线）
    |
    v
谈判/交易（条件开价、讨价还价）
    |
    v
结果（成交/破裂/第三方搅局）
    |
    v
后果发酵（赢家付出什么代价，输家留下什么隐患）
```

### 暴力规则

- 暴力是谈判失败后的最后手段，不是第一选择
- 动手前要有"为什么非打不可"的逻辑
- 暴力后果必须落地：伤亡、仇恨、警方介入、势力变化
- 不要无差别暴力，要精准暴力

---

## 主角约束（如适用）

- 极致利己，但不是无脑疯狗
- 讲收益、讲底线、讲代价，做事有算计
- 需要保留"非功能性时刻"：抽烟、失眠、沉默、试探
- 成功伴随不可逆代价，表层是爽点，里层是成本
- 斩草除根但有底线，底线由利益决定不由道德决定

---

## 年代硬约束

| 项目 | 约束 |
|------|------|
| 通讯 | 80 年代传呼机/座机，90 年代才有大哥大和早期手机 |
| 交通 | 具体车型要对应年代，不能穿越 |
| 物价 | 参考当年实际物价，重要金额需考据 |
| 法律 | 殖民时期法律体系，97 之前和之后区分 |
| 社会 | 当年的社会事件、流行文化、社会氛围 |
| 地理 | 街道名、建筑物、区域特征要符合当年 |

真实人物与敏感实体优先用代指或化名。
