# 港综禁止模式

> 港综题材特有的负面清单。与 base-rules.md 中的全局禁止模式叠加使用。

---

## 叙事禁止

| 禁止模式 | 说明 | 替代方案 |
|----------|------|----------|
| 历史课件开头 | 不以年代背景介绍开场 | 用场景/动作/交易/压迫感切入 |
| 空心装狠 | 不靠喊口号体现角色强大 | 用行动、物件、价格、制度摩擦说话 |
| 无脑种马 | 不写无逻辑的感情线 | 感情线必须服务于利益或剧情 |
| 原著复读机 | 不照搬参考作品的桥段 | 只借鉴结构和节奏，不搬情节 |
| 系统保姆化 | 辅助系统不替主角解决问题 | 系统提供信息/工具，决策靠主角 |
| 连续碾压 | 不把每章都写成主角无敌模板 | 成功伴随代价，偶尔受阻 |

## 角色禁止

| 禁止模式 | 说明 | 替代方案 |
|----------|------|----------|
| 配角三功能 | 配角只会震惊、附和、送人头 | 给配角自己的算盘、恐惧、筹码 |
| 反派木桩 | 反派排队送死 | 反派有试探、误判、反扑 |
| 主角心软 | 为推剧情突然仁慈讲武德 | 留人必须是利益更大，绝不是心软 |
| 信息越界 | 反派使用不可能知道的信息 | 反派行动追溯到已知信息和逻辑 |
| 无脑疯狗 | 主角不计后果地杀戮 | 极致利己但有算计，讲收益讲代价 |

## 语言禁止

| 禁止模式 | 说明 | 替代方案 |
|----------|------|----------|
| 港味堆砌 | 为了港味每句都塞粤语 | 适度使用，上下文让读者理解 |
| 通用狠话 | 所有角色说一样的威胁 | 台词贴身份、年代、阶层、关系 |
| 空话循环 | 连续同义反复表达同一意思 | 每段带来新信息或态度变化 |
| 模板化台词 | "你知道我是谁吗""给你三天时间" | 根据具体情境和角色定制 |

## 设定禁止

| 禁止模式 | 说明 | 替代方案 |
|----------|------|----------|
| 年代穿越 | 80 年代出现智能手机 | 通讯、交通、物价严格对标年代 |
| 地理穿帮 | 虚构不存在的香港地名/建筑 | 使用真实地名，必要时联网考据 |
| 法律混淆 | 混用殖民时期和回归后法律 | 区分 97 前后法律体系 |
| 物价失真 | 金额不符合当年物价水平 | 重要金额需考据 |
| 敏感直写 | 直接使用真实人物全名 | 用代指或化名 |

---

## 数据禁止

| 禁止模式 | 说明 | 替代方案 |
|----------|------|----------|
| 模糊收益 | "赚了一大笔" | 具体金额和条件 |
| 数据通胀 | 资产无逻辑暴涨 | 每笔收入有来源和逻辑 |
| 利益链断裂 | 前文的交易后文无交代 | 交易必须有后续兑现或变故 |
